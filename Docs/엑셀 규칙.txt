6. Row 2

7. Row 3
-> 각 Column의 자료형
-> int, float, string, int[], float[], string[], enum, enum:EType, class:DataType, enum[]:EType, class[]:DataType, noti 등으로 구성
-> [] << 배열로 구성되는 타입
-> 배열의 경우 네이밍이 동일해야 함
예) int[]로 가정. (같은 이름이여야 하나의 배열로 묶음)

-> class:DataType : DataType은 Excel Data와 매칭. 예) class:DataBalance
-> noti : 주석용으로 툴에서 읽지 않음
